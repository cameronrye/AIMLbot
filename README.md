## Installation
```sh
npm install
```

## Commandline
```sh
npm run cli
```

## Usage
```sh
Usage
    aimlbot [<file|directory> ...]

Options
    --help                 This help screen

Examples
    aimlbot
    aimlbot bot.aiml
    aimlbot /aiml
```

## Development Setup
You will get all of your JS files linted by Standard.js and tests run by Ava

```sh
npm run dev
```
